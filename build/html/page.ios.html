class page.ios.home_page.HomePage(driver: appium.webdriver.webdriver.WebDriver)[源代码]¶

基类：page.ios_common_page.IosCommonPage

页面类示例 页面类一般推荐继承对应的common_page，

common_page主要用于公共页面元素和公共方法的封装，所有页面共用

元素示例： 由定位方式，定位的值，元素信息组成， locator = (By.ID, ‘element_id’, “元素信息”) 元素信息可选，主要用于打印调试日志以及方便后期维护元素 或 locator = (By.ID, ‘element_id’)
